## props

参数	| 说明	| 类型	| 默认值	| 可选值

wrapStyle | 组件的style | string | - | -
wrapClass | 组件的class | string | - | -
viewClass | 内容区域的class | string | - | -

native | 是否取消展示滚动条 | boolean | false | -
