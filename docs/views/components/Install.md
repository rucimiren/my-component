# 安装

## install

```js
npm install @rucimiren/xiao-ui -S
```

## import

```js
//在入口文件(一般是main.js)引入
import xiao from '@rucimiren/xiao-ui'
import '@rucimiren/xiao-ui/lib/xiao.css'

Vue.use(xiao)
```
